## 将棋ライブラリの説明
将棋盤オブジェクトを作成し、将棋ゲームを数行で作成できるようにしようとしました。


駒の表記についてはsfen形式に準拠しています
例:
p:後手の歩 
L:先手の香車


```test.py
if __name__ == '__main__':

    # sfen形式を引数にShogiBoardオブジェクトを作成できます
    _sfen = "sfen lnsgkgsnl/1r5b1/ppppppppp/9/9/9/PPPPPPPPP/1B5R1/LNSGKGSNL b - 1"

    sample_board.show(turn_label="●")  # 任意のマークで手番を示すオプションあり。デフォルトは"【手番】"

    move = "７六歩(77)"  # kif形式の動きをサポートしています。
```
